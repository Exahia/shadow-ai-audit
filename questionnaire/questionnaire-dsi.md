# Questionnaire Shadow AI — Pour DSI / RSSI

Questionnaire à diffuser aux responsables de département pour évaluer l'ampleur du Shadow AI.

## Instructions
- Diffuser de manière **anonyme** pour obtenir des réponses honnêtes
- Cibler : managers, chefs de projet, responsables d'équipe
- Durée : 5-10 minutes

---

## Section 1 : Usage actuel de l'IA

1. **Utilisez-vous des outils d'IA au travail ?**
   - [ ] Oui, quotidiennement
   - [ ] Oui, plusieurs fois par semaine
   - [ ] Oui, occasionnellement
   - [ ] Non

2. **Quels outils utilisez-vous ?** (plusieurs réponses possibles)
   - [ ] ChatGPT (OpenAI)
   - [ ] Microsoft Copilot
   - [ ] Google Gemini
   - [ ] Claude (Anthropic)
   - [ ] Autre : ___

3. **Ces outils sont-ils approuvés par la DSI ?**
   - [ ] Oui, officiellement
   - [ ] Je ne sais pas
   - [ ] Non
   - [ ] Il n'y a pas de politique IA dans mon entreprise

4. **Pour quelles tâches utilisez-vous l'IA ?** (plusieurs réponses)
   - [ ] Rédaction (emails, rapports, comptes-rendus)
   - [ ] Résumé de documents
   - [ ] Traduction
   - [ ] Analyse de données
   - [ ] Code / développement
   - [ ] Recherche d'information
   - [ ] Autre : ___

## Section 2 : Données exposées

5. **Quels types de données avez-vous déjà partagés avec un outil IA ?**
   - [ ] Données internes (stratégie, projets, roadmap)
   - [ ] Données clients (noms, contacts, contrats)
   - [ ] Données financières (bilans, budgets, prévisions)
   - [ ] Données RH (salaires, évaluations, candidatures)
   - [ ] Données techniques (code source, architecture, brevets)
   - [ ] Aucune donnée sensible

6. **Avez-vous conscience que les données envoyées à ChatGPT sont stockées sur des serveurs américains soumis au Cloud Act ?**
   - [ ] Oui
   - [ ] Non
   - [ ] Je ne sais pas ce qu'est le Cloud Act

## Section 3 : Besoins

7. **Si votre entreprise proposait un outil IA interne et sécurisé, l'utiliseriez-vous ?**
   - [ ] Oui, immédiatement
   - [ ] Oui, si c'est aussi performant que ChatGPT
   - [ ] Peut-être
   - [ ] Non, je préfère ChatGPT

8. **Quelles fonctionnalités seraient prioritaires ?**
   - [ ] Chat (questions/réponses)
   - [ ] Résumé de documents
   - [ ] Recherche dans les documents internes (RAG)
   - [ ] Traduction
   - [ ] Génération de code
   - [ ] Autre : ___

---

## Exploitation des résultats

Ce questionnaire permet de :
1. **Quantifier** le Shadow AI (% d'utilisateurs non autorisés)
2. **Identifier** les données exposées par département
3. **Prioriser** les départements à équiper en premier
4. **Justifier** l'investissement dans une solution souveraine

Voir le [template de rapport](../rapport/template-rapport.md) pour structurer les résultats.
